# SOP · Пакет «Карты + контент» — 20 000 ₽/мес

Скоуп: актуальная карточка Яндекс Бизнес, ответы на отзывы ≤ 48 ч, 4–6 публикаций/мес (канал зафиксирован при онбординге). НЕ входит: накрутка отзывов (не делаем никогда), съёмка на месте.

## Онбординг
- Доступ к Яндекс Бизнес клиента, аудит карточки: категории, услуги с ценами, фото, график, атрибуты.
- Зафиксировать канал публикаций (сайт / соцсеть) и темы рубрик с клиентом.

## Еженедельно
- Отзывы: новые → черновик ответа (негатив: признать, предложить решение, увести в личку) → Review при негативе, позитив — отвечать сразу. SLA ≤ 48 ч.
- Карточка: цены/услуги актуальны? Изменения от клиента — внести.

## Ежемесячно
1. 4–6 публикаций: контент-план начала месяца → тексты (скилл `deaify`) → Review → публикация по графику.
2. Фото: запросить у клиента свежие (работы, команда) — минимум 2 новых в карточку.
3. Отчёт: просмотры/действия из Яндекс Бизнес, рейтинг, отзывы (сколько, средняя оценка), публикации. → клиенту.
4. Счёт: `bill.py create {клиент} --amount 20000` → подтверждение владельца.

## Правила
- На агрессивный/несправедливый отзыв ответ — только через Review владельца.
- Скидки/акции в публикациях — только подтверждённые клиентом письменно.
