# SOP · Пакет «Сайт+» — 25 000 ₽/мес

Скоуп: до 10 задач-правок/мес (правка = до 1 часа работы), мониторинг, бэкапы, контент по материалам клиента. НЕ входит: новые разделы, редизайн, интеграции, миграции → шаблон №4 (допсоглашение).

## Постоянно
- Заявка клиента → issue на борд (Inbox) в течение часа → классификация: правка ≤1ч → Todo; больше → допсмета.
- SLA правки: ≤ 48 ч до Done. Считать правки месяца: 8-я из 10 → предупредить клиента об остатке лимита.

## Еженедельно
- Healthcheck: сайт отвечает (curl 200), SSL не истекает (< 21 дня — перевыпуск), формы отправляются.
- Проверить бэкап: свежий дамп/снапшот существует.

## Ежемесячно (1-е число)
- Отчёт клиенту: список закрытых правок (из борда, фильтр по репо за месяц), uptime, что рекомендуем.
- Счёт: `bill.py create {клиент} --amount 25000` → подтверждение владельца → отправка. Акт — по закрытии месяца.

## Границы
- Деплой на прод — только после Review. Правки текста/фото — без ревью, деплой сразу.
- Просьба «а сделайте ещё и...» вне скоупа — не делать «в счёт», шаблон №4.
