# SOP · Пакет «SEO-абонентка» — 35 000 ₽/мес · минимум 3 месяца

Скоуп: техоптимизация, 2–4 новые/обновлённые страницы/мес, 2 SEO-текста/мес, отчёт. НЕ входит: ссылочные бюджеты, гарантии позиций, контекст.

## Месяц 1 (онбординг)
- Аудит: `yandex-webmaster` (ошибки индексации, исключённые страницы), скорость (PageSpeed), разметка, sitemap/robots.
- Семантика: `yandex-wordstat` по услугам/товарам клиента → карта запросов → структура страниц. Приоритеты клиента (как список Филиппа) — в начало очереди.
- План на квартал → согласование с клиентом.

## Ежемесячный цикл
1. Неделя 1: техзадачи месяца (из аудита) + выбор 2–4 страниц из карты запросов.
2. Недели 2–3: тексты (2 шт: скилл `deaify` — снять AI-стиль), новые/обновлённые страницы, внутренняя перелинковка.
3. Неделя 4: отчёт — позиции (вебмастер), поисковый трафик (`yandex-metrika`), сделано/планы. Отчёт → Review → клиенту.
4. Счёт: `bill.py create {клиент} --amount 35000` → подтверждение владельца.

## Правила
- До 3-го месяца не обещать динамику позиций — только объём выполненного.
- Любые «давайте закупим ссылок» — вне пакета, шаблон №4.
- Тексты публикуются только после Review (факты о компании клиента проверить).
