# Руководство по адаптивности для Desktop разрешений

## Обзор изменений

Сайт полностью адаптирован для всех экранов ПК (1280px, 1440px, 1680px, 1920px, 2560px и выше) с использованием современных fluid layout подходов.

## Ключевые принципы реализации

### 1. Fluid Layout (Гибкая вёрстка)

**Что сделано:**
- Все контейнеры используют `flex` или `grid` вместо фиксированных ширин
- Grid layouts с адаптивными `minmax()` значениями
- Автоматическое масштабирование колонок на больших экранах

**Примеры:**
```css
/* До */
.grid-3 {
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
}

/* После */
.grid-3 {
  grid-template-columns: repeat(auto-fit, minmax(clamp(280px, 30vw, 380px), 1fr));
}
```

### 2. Адаптивная типографика с clamp()

**Что сделано:**
- Все размеры шрифтов используют `clamp()` для плавного масштабирования
- Формула: `clamp(min, preferred, max)` где preferred использует vw + rem
- Добавлены `word-wrap` и `overflow-wrap` для предотвращения переполнения

**Примеры:**
```css
/* h1 - главные заголовки */
font-size: clamp(2.5rem, 4vw + 1rem, 5rem);

/* h2 - подзаголовки */
font-size: clamp(2rem, 3vw + 0.5rem, 3.5rem);

/* body text */
font-size: clamp(1rem, 1.1vw + 0.125rem, 1.125rem);
```

### 3. Fluid spacing (Адаптивные отступы)

**Что сделано:**
- CSS переменные обновлены для использования clamp()
- Padding и margins масштабируются пропорционально размеру экрана
- Специальные media queries для очень больших экранов (2560px+)

**CSS переменные:**
```css
:root {
  --spacing-xs: clamp(0.5rem, 0.5vw, 0.75rem);
  --spacing-sm: clamp(1rem, 1vw, 1.25rem);
  --spacing-md: clamp(1.5rem, 2vw, 2rem);
  --spacing-lg: clamp(3rem, 4vw, 5rem);
  --spacing-xl: clamp(5rem, 6vw, 8rem);
  --spacing-2xl: clamp(7rem, 8vw, 10rem);
}
```

### 4. Адаптивная максимальная ширина

**Что сделано:**
- `max-width` контейнеров использует clamp() вместо фиксированных значений
- Контент растягивается до 75% ширины viewport на больших экранах
- Разные варианты контейнеров для разного типа контента

**Примеры:**
```css
/* Основной контейнер */
--max-width: clamp(1200px, 75vw, 1600px);

/* Текстовый контейнер */
--max-width-text: clamp(900px, 60vw, 1200px);

/* Узкий контейнер */
--max-width-narrow: clamp(700px, 50vw, 900px);
```

### 5. Предотвращение горизонтального скролла

**Что сделано:**
- `overflow-x: hidden` на body
- `width: 100%` и `box-sizing: border-box` на всех основных контейнерах
- `white-space: nowrap` на кнопках и тегах
- `word-wrap` и `overflow-wrap` на всех текстовых элементах

### 6. Адаптивные Grid-системы

**Обновлённые grid layouts:**
```css
/* 3-колоночная сетка */
.grid-3 {
  grid-template-columns: repeat(auto-fit, minmax(clamp(280px, 30vw, 380px), 1fr));
  gap: clamp(1.5rem, 2vw, 2.5rem);
}

/* 2-колоночная сетка */
.grid-2 {
  grid-template-columns: repeat(auto-fit, minmax(clamp(280px, 35vw, 450px), 1fr));
  gap: clamp(1.5rem, 2vw, 2.5rem);
}
```

## Особенности для разных разрешений

- Контент занимает ~75% ширины экрана
- Комфортная читаемость текста
- Сбалансированное расположение элементов

### 1680px - 1920px (Full HD мониторы)
- Увеличенные отступы для лучшего восприятия

- Расширенные контейнеры до 1400-1500px

- Максимальная ширина контента до 1600px
- Увеличенные размеры шрифтов и элементов
- Дополнительные отступы для предотвращения растягивания

## Технические детали

### Использование clamp()
```css
/* Синтаксис */
clamp(минимум, предпочтительное_значение, максимум)

/* Предпочтительное значение обычно включает vw */
clamp(1rem, 1vw + 0.125rem, 1.25rem)

/* Расшифровка: */
/* - Минимум: 1rem (16px) */
/* - Предпочтительное: 1vw + 0.125rem (масштабируется с viewport) */
/* - Максимум: 1.25rem (20px) */
```

### Viewport units (vw, vh)
- `1vw` = 1% ширины viewport
- Используется в комбинации с `rem` для fluid typography
- Пример: `2vw + 0.5rem` даёт плавное масштабирование с базовым размером

### Box-sizing
```css
* {
  box-sizing: border-box; /* Padding включён в width */
}
```

### Word breaking
```css
.text-element {
  word-wrap: break-word;        /* Legacy поддержка */
  overflow-wrap: break-word;    /* Современный стандарт */
}
```

## Fallback для старых браузеров

Если требуется поддержка старых браузеров, можно добавить fallback:

```css
.element {
  font-size: 1.125rem;                          /* Fallback */
  font-size: clamp(1rem, 1.1vw + 0.125rem, 1.25rem); /* Modern */
}
```

## Дополнительные media queries

Для особых случаев добавлены специальные breakpoints:

```css
/* Большие экраны */
@media (min-width: 1920px) {
  .container {
    padding: 0 clamp(2rem, 4vw, 4rem);
  }
}

/* 4K и выше */
@media (min-width: 2560px) {
  .container {
    padding: 0 clamp(3rem, 5vw, 6rem);
  }
  
  .grid-3 {
    grid-template-columns: repeat(auto-fit, minmax(450px, 1fr));
  }
}
```

## Поддержка и вопросы

При возникновении проблем с адаптивностью:
1. Проверьте консоль браузера на ошибки
2. Убедитесь, что новые элементы используют fluid подход
3. Проверьте, что `box-sizing: border-box` применён
4. Убедитесь в отсутствии фиксированных ширин в px
