Раньше создание договора занимало 30-40 минут. Открыть шаблон, заполнить реквизиты, проверить, экспортировать в PDF, загрузить в Saby, заполнить поля, подписать ЭЦП, отправить.

Теперь я пишу в Telegram:

> Договор для [клиент] на автоматизацию

И через 15 секунд получаю ссылку на готовый документ в Saby. Осталось нажать «Подписать».

## Как это работает

AI-ассистент делает всё сам:

1. **Находит данные клиента** — ИНН, КПП, адрес, банковские реквизиты. Всё хранится в структурированных карточках.

2. **Генерирует договор** — заполняет шаблон: предмет, сумма, сроки, порядок оплаты. По ГК РФ, с нужными пунктами.

3. **Отправляет в Saby** — через API. Не просто загружает PDF, а создаёт документ с метаданными: сумма, контрагент, период действия.

4. **Присылает ссылку** — открываю, проверяю, подписываю ЭЦП. Один клик.

## Техническая часть

Интеграция с Saby через их JSON-RPC API:

\`\`\`
POST https://online.sbis.ru/service/?srv=1
{
  "method": "СБИС.ЗаписатьДокумент",
  "params": {
    "Документ": {
      "Тип": "ДоговорИсх",
      "Номер": "...",
      "Сумма": "...",
      "Контрагент": { "СвЮЛ": { "ИНН": "..." } }
    }
  }
}
\`\`\`

AI аутентифицируется, создаёт документ, прикрепляет PDF, заполняет все поля. Saby принимает — документ готов к подписанию.

## Что изменилось

**Было:** 30-40 минут на договор. Копипаст реквизитов, ручное заполнение, риск ошибок.

**Стало:** 15 секунд + проверка. AI не путает ИНН, не забывает КПП, не ошибается в суммах.

## Ограничения

Подпись — только вручную. ЭЦП требует подтверждения человеком. Это правильно: AI готовит документ, человек принимает решение.

Шаблоны — пока базовые. Для сложных договоров нужна доработка.
