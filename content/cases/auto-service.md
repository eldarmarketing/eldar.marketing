Комплексный интернет-маркетинг для сети автосервисов: SEO, контекст, аналитика.

## Задача

Увеличить поток клиентов в 3 филиала автосервиса. Текущая ситуация: сайт есть, но заявок мало.

## Что сделали

**SEO-оптимизация**

- Переработали структуру сайта под поисковые запросы
- Создали страницы под каждую услугу
- Настроили локальное SEO для каждого филиала

**Контекстная реклама**

- Яндекс Директ: поиск + РСЯ
- Telegram Ads
- Ретаргетинг на посетителей сайта

**Аналитика**

- Настроили сквозную аналитику
- Коллтрекинг для отслеживания звонков
- Еженедельные отчёты

## Результаты

- **+180%** заявок с сайта
- **-35%** стоимость заявки
- **ТОП-5** по основным запросам в регионе
