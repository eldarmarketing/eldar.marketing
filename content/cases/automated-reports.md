## Проблема

Владелец автоателье в&nbsp;Санкт-Петербурге каждый день задавала одни и&nbsp;те&nbsp;же вопросы:

— Сколько людей зашло на&nbsp;сайт?
— Откуда трафик&nbsp;— реклама или поиск?
— Сколько звонков было? Сколько пропустили?

Ответы приходилось собирать вручную: зайти в&nbsp;Метрику, открыть личный кабинет телефонии, посчитать. На&nbsp;это уходило 10–15&nbsp;минут каждый день. А&nbsp;если забыл&nbsp;— картина за&nbsp;день терялась.

## Задача

Сделать так, чтобы владелец получала полную сводку по&nbsp;бизнесу каждое утро в&nbsp;Telegram. Без необходимости заходить в&nbsp;Метрику, телефонию или любой другой сервис.

## Что подключили

### 1. Яндекс.Метрика

Через API Метрики бот забирает данные за&nbsp;вчерашний день:

- **Посетители и&nbsp;визиты**&nbsp;— сколько людей было на&nbsp;сайте
- **Процент отказов**&nbsp;— сколько ушли сразу
- **Глубина просмотра**&nbsp;— сколько страниц в&nbsp;среднем смотрят
- **Время на&nbsp;сайте**&nbsp;— насколько контент удерживает
- **Источники трафика**&nbsp;— реклама, поиск, прямые заходы, соцсети
- **Топ страниц**&nbsp;— какие страницы смотрят чаще всего (с&nbsp;кликабельными ссылками)

Один запрос к&nbsp;API&nbsp;— и&nbsp;все цифры в&nbsp;сообщении.

### 2. Телефония (Sipuni)

Автоателье принимает много звонков&nbsp;— клиенты узнают цены, записываются на&nbsp;работы. Через API телефонии бот собирает:

- **Общее количество звонков** за&nbsp;день
- **Входящие и&nbsp;исходящие** отдельно
- **Принятые и&nbsp;пропущенные**&nbsp;— главный показатель для&nbsp;владельца
- **Средняя длительность разговора**&nbsp;— косвенный показатель качества консультации

Пропущенный звонок&nbsp;— это потенциально потерянный клиент. Теперь владелец видит это каждое утро.

### 3. Речевая аналитика (Yandex SpeechSense)

Следующий уровень&nbsp;— анализ содержания разговоров. Все звонки через телефонию автоматически отправляются в&nbsp;SpeechSense, который:

- **Транскрибирует** каждый разговор в&nbsp;текст
- **Определяет настроение** клиента (доволен, нейтрален, недоволен)
- **Оценивает качество** обслуживания оператором
- **Выделяет ключевые темы**&nbsp;— о&nbsp;чём спрашивают чаще всего
- **Считает соотношение** речи оператора и&nbsp;клиента

Это даёт владельцу понимание не&nbsp;только «сколько звонков», но&nbsp;и&nbsp;«о&nbsp;чём звонят» и&nbsp;«довольны&nbsp;ли клиенты».

## Как выглядит отчёт

Каждое утро в&nbsp;09:00 в&nbsp;Telegram приходит одно сообщение:

```
📊 Отчёт за 25.02.2026

👥 Посетители: 47
👁 Визиты: 66
📉 Отказы: 29%
📄 Глубина: 3.4 стр.
⏱ Время на сайте: 5:13

📡 Источники:
  Прямые заходы: 36 (22 чел.)
  Реклама: 20 (19 чел.)
  Поиск: 10 (9 чел.)

📞 Звонки:
  Всего: 19 (📥 7 вх. / 📤 12 исх.)
  ✅ Принято: 15 / ❌ Пропущено: 4
  ⏱ Средний разговор: 1:02
```

Никаких дашбордов, логинов, паролей. Открыл Telegram&nbsp;— увидел картину дня.

## Техническая реализация

- **Node.js скрипт** на&nbsp;сервере (~120&nbsp;строк)
- **Cron** запускает его раз в&nbsp;сутки в&nbsp;09:00
- **Яндекс.Метрика API**&nbsp;— 3&nbsp;запроса (статистика, источники, страницы)
- **Sipuni API**&nbsp;— 1&nbsp;запрос (CSV со&nbsp;звонками за&nbsp;день)
- **Telegram Bot API**&nbsp;— отправка сообщения
- **SpeechSense**&nbsp;— автоматическая интеграция через Sipuni, без&nbsp;дополнительного кода

Всё работает на&nbsp;одном сервере. Никаких внешних сервисов, подписок или no-code платформ.

## Результат

| До | После |
|-----|-------|
| Ручная проверка 3&nbsp;сервисов | Один отчёт в&nbsp;Telegram |
| 10–15&nbsp;минут каждый день | 0&nbsp;минут |
| Забыл проверить&nbsp;— данные потеряны | Автоматически каждое утро |
| Только цифры посещений | + звонки + аналитика разговоров |

Владелец тратит 30&nbsp;секунд на&nbsp;чтение сводки вместо 15&nbsp;минут на&nbsp;сбор данных. А&nbsp;с&nbsp;речевой аналитикой появилось понимание качества работы с&nbsp;клиентами, которого раньше не&nbsp;было вообще.

## Сколько это стоит

- **Яндекс.Метрика API**&nbsp;— бесплатно
- **Sipuni API**&nbsp;— входит в&nbsp;тариф телефонии
- **SpeechSense**&nbsp;— от&nbsp;3,2&nbsp;₽ за&nbsp;минуту разговора (~60&nbsp;₽/день при&nbsp;20&nbsp;звонках)
- **Разработка**&nbsp;— 2&nbsp;часа работы AI-ассистента

Итого: практически бесплатно при&nbsp;уже подключенной телефонии.
