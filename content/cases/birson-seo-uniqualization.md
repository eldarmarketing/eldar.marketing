У автоателье BIRSON на сайте 28 000 страниц — каждая комбинация услуги и модели авто. Перетяжка руля BMW X5, перетяжка салона Mercedes E-Class, восстановление airbag Audi Q7 — и так далее. Яндекс индексировал из них 230. Остальные выглядели одинаково — один текст, подставленный бренд.

## Задача

Сделать так, чтобы Яндекс начал индексировать модельные страницы. Для этого — уникализировать контент на самых ценных из них.

## Что было

- 27 услуг и 1 036 моделей авто — 28 000 комбинаций
- Один шаблон на все: заголовок "Перетяжка руля {brand} {model}", один и тот же текст, одинаковые FAQ
- Яндекс видит дублирование — индексирует меньше 1%
- Потенциал programmatic SEO не используется — тысячи длиннохвостовых запросов без ответа

## Что сделали

### Шаг 1. Выбрали 100 ключевых страниц

20 самых востребованных моделей (BMW X5, Mercedes E-Class, Audi Q5, Toyota Camry, Chery Tiggo 7 Pro и другие) умножили на 5 топ-услуг (руль, салон, сиденья, потолок, airbag). Получили 100 страниц — самые ценные по SEO.

### Шаг 2. Сгенерировали уникальные тексты

Для каждой из 100 страниц — свой контент:
- Заголовок с названием модели и услуги
- Текст на 3-5 предложений — специфика работы с конкретной моделью, коды кузовов (G05, W213, FY), реальные сроки и цены
- 3 конкретных преимущества — не "высокое качество", а "сохраняем подогрев и кнопки мультимедиа"
- 4 FAQ — сформулированы как реальные поисковые запросы ("сколько стоит перетяжка руля BMW X5 в СПб")

Тексты написаны как от мастера, который работает с этими машинами каждый день. Без "индивидуального подхода" и "команды профессионалов".

### Шаг 3. Обновили шаблон страницы

Страница проверяет — есть ли для данной пары (услуга + модель) уникальный контент. Если есть — показывает его. Если нет — работает старый шаблон. Ничего не сломано.

Дополнительно на каждую модельную страницу добавили:
- Секция "Наши последние работы" — реальные фото из портфолио
- Секция "Отзывы наших клиентов" — настоящие отзывы
- Уникальные meta title и description — для сниппетов в поиске
- FAQ Schema — для расширенных сниппетов в Яндексе и Google

### Шаг 4. Деплой

Генерация 100 текстов, интеграция в шаблон и деплой на продакшен — за одну сессию Claude Code.

## Результат

| Метрика | Было | Стало |
|---------|------|-------|
| Уникальных страниц | 0 из 28 000 | 100 из 28 000 |
| Контент на странице | Один шаблон с {brand} | Уникальный текст, FAQ, портфолио, отзывы |
| Meta description | Одинаковый на всех | Уникальный на 100 страницах |
| FAQ Schema | Шаблонные 4 вопроса | Уникальные поисковые запросы |

Индексация в Яндексе — ждём. Обычно уникальные страницы начинают появляться в индексе через 2-4 недели.
