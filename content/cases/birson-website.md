## Задача

Клиент&nbsp;— автоателье BIRSON, специализируется на&nbsp;перетяжке салонов, покраске кожи и&nbsp;предпродажной подготовке автомобилей в&nbsp;Санкт-Петербурге. Два направления: премиальная перетяжка для&nbsp;владельцев и&nbsp;быстрая подготовка для&nbsp;перекупщиков.

Нужен был современный сайт, который передаёт премиальность услуг, показывает портфолио работ и&nbsp;конвертирует посетителей в&nbsp;заявки.

## Решение

Стек: Next.js&nbsp;16 + React&nbsp;19 + Tailwind&nbsp;CSS + Prisma + PocketBase + Grammy (Telegram-бот).

### Дизайн

- Тёмная тема с&nbsp;золотыми акцентами&nbsp;— премиальный вид, ассоциация с&nbsp;кожей и&nbsp;люксом
- Фоновое видео/фото текстуры кожи на&nbsp;hero-секции
- Mobile-first: адаптивная верстка для&nbsp;всех устройств
- CTA «Рассчитать стоимость» в&nbsp;шапке

### Структура (10+ страниц)

- Главная с&nbsp;hero-секцией, преимуществами и&nbsp;блоком для&nbsp;перекупщиков
- Услуги с&nbsp;детальными страницами по&nbsp;каждому виду работ
- Калькулятор стоимости&nbsp;— онлайн-расчёт без&nbsp;визита
- Портфолио с&nbsp;фото работ
- Цены
- Отзывы
- Блог
- Контакты с&nbsp;Яндекс.Картой
- Админ-панель
- Политика конфиденциальности

![Услуги](/images/cases/birson/services.png)

### Фичи

- **Калькулятор стоимости**&nbsp;— клиент получает цену без&nbsp;звонка
- **Блок для&nbsp;перекупщиков**&nbsp;— отдельная воронка: «Сфоткай → Отправь в&nbsp;Telegram → Получи цену за&nbsp;10&nbsp;минут»
- **ROI-калькулятор**&nbsp;— «Вложи 5&nbsp;000&nbsp;₽ → получи +20&nbsp;000&nbsp;₽ к&nbsp;цене авто»
- **Telegram-бот**&nbsp;— CRM для&nbsp;заявок и&nbsp;коммуникации
- **Портфолио через Telegram**&nbsp;— мастер отправляет фото в&nbsp;бот, они появляются на&nbsp;сайте
- **S3-хранилище** для&nbsp;медиафайлов
- **Генерация PDF** (счета, реквизиты)
- JSON-LD Schema для&nbsp;SEO
- Яндекс.Метрика

## Результат

- Современный премиальный сайт, передающий качество услуг
- Две воронки: для&nbsp;владельцев авто и&nbsp;для&nbsp;перекупщиков
- Калькулятор снижает нагрузку на&nbsp;менеджера
- Telegram-бот автоматизирует приём заявок и&nbsp;публикацию портфолио
- Сайт работает на&nbsp;VPS (PM2 + Nginx + SSL)

## Для&nbsp;кого

Автоателье, детейлинг-студии, автосервисы с&nbsp;премиальным позиционированием, бизнесы которым нужен калькулятор стоимости на&nbsp;сайте.
