Разработка Telegram Mini App для P2P-обмена криптовалюты.

## Задача

Создать удобный интерфейс для обмена USDT внутри Telegram. Без перехода на внешние сайты.

## Решение

**Mini App**

- Интерфейс обменника внутри Telegram
- Актуальные курсы в реальном времени
- История операций
- Уведомления о статусе

**Интеграции**

- Подключение к биржевым API
- Автоматический расчёт курса
- Верификация пользователей

## Результаты

- **500+** пользователей за первую неделю
- **< 2 сек** время загрузки
- **0** жалоб на UX
