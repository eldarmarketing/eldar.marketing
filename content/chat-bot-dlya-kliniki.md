Половина обращений в&nbsp;клинику приходит вечером и&nbsp;в&nbsp;выходные — когда администратор не&nbsp;на&nbsp;месте. Пропущенный звонок почти всегда означает: пациент записался к&nbsp;конкуренту через дорогу. Чат-бот закрывает эту дыру — принимает заявки и&nbsp;записывает на&nbsp;приём в&nbsp;любое время суток.

## Что делает бот для клиники

- **Записывает на&nbsp;приём 24/7** — показывает свободные окна, фиксирует врача, услугу и&nbsp;время без участия администратора
- **Напоминает о&nbsp;визите** — сообщение за&nbsp;1–2&nbsp;дня и&nbsp;за&nbsp;несколько часов до&nbsp;приёма снижает долю no-show
- **Отвечает на&nbsp;частые вопросы** — цены, адрес, график работы, что взять с&nbsp;собой, действует ли&nbsp;полис ДМС
- **Переносит и&nbsp;отменяет запись** — пациент меняет время сам, без звонка администратору
- **Интегрируется с&nbsp;МИС и&nbsp;CRM** — работаем через API с&nbsp;IDENT, Medesk, Клиентикс, 1С:Медицина, Renovatio. Карточка пациента и&nbsp;история приёмов обновляются автоматически
- **Хранит данные в&nbsp;России** — требование 152-ФЗ для&nbsp;медицинских данных. Бот не&nbsp;ставит диагнозы и&nbsp;не&nbsp;даёт медицинских рекомендаций — только запись и&nbsp;организационные вопросы

## Часто задаваемые вопросы

### Как бот работает с&nbsp;медицинскими данными и&nbsp;152-ФЗ?

Данные пациентов хранятся на&nbsp;серверах в&nbsp;России, собираются только с&nbsp;согласия, доступ ограничен. Бот не&nbsp;ставит диагнозы и&nbsp;не&nbsp;даёт медицинских рекомендаций — только запись, напоминания и&nbsp;организационные вопросы.

### Интегрируется ли&nbsp;бот с&nbsp;нашей МИС?

Работаем через API с&nbsp;популярными системами: IDENT, Medesk, Клиентикс, 1С:Медицина, Renovatio. Если у&nbsp;МИС есть API — интеграция возможна. Если нет — строим контур записи рядом, без замены МИС.

### Сколько времени занимает внедрение?

Простой бот с&nbsp;записью и&nbsp;FAQ — 3–5&nbsp;дней. Бот с&nbsp;интеграцией МИС и&nbsp;CRM — 1–2&nbsp;недели. Сложная система с&nbsp;базой знаний клиники — 2–4&nbsp;недели.

### Насколько снижается no-show?

Автоматические напоминания за&nbsp;1–2&nbsp;дня и&nbsp;за&nbsp;несколько часов до&nbsp;приёма — стандартная практика, снижающая долю неявок в&nbsp;разы. Точную цифру считаем на&nbsp;данных конкретной клиники после 1–2&nbsp;месяцев работы бота.
