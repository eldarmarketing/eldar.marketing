Делаем IT-систему для клиник, стоматологий и&nbsp;медицинских центров: AI-администратор, сайт с&nbsp;онлайн-записью, CRM, реклама и&nbsp;аналитика. Не&nbsp;набор подрядчиков, а&nbsp;один контур, где каждый пропущенный звонок и&nbsp;каждая заявка учтены. Работаем по&nbsp;всей России.

## Что болит у&nbsp;клиник

**Пропущенные звонки = потерянные пациенты.** Администратор занят на&nbsp;ресепшене, звонок ушёл в&nbsp;никуда. Пациент не&nbsp;перезванивает — записывается к&nbsp;конкуренту через дорогу.

**No-show съедает расписание.** Пациент записался и&nbsp;не&nbsp;пришёл. Кресло простаивает, врач без загрузки, деньги потеряны.

**Запись только в&nbsp;рабочие часы.** Половина обращений приходит вечером и&nbsp;в&nbsp;выходные — когда ответить некому.

**Реклама без аналитики.** Бюджет уходит в&nbsp;Директ и&nbsp;агрегаторы, но&nbsp;непонятно, какой канал приводит первичных пациентов, а&nbsp;какой сливает деньги.

## Что внедряем

### AI-администратор 24/7

Виртуальный сотрудник в&nbsp;Telegram, на&nbsp;сайте и&nbsp;в&nbsp;мессенджерах:

- Записывает на&nbsp;приём в&nbsp;свободные слоты — ночью, в&nbsp;выходные, в&nbsp;праздники
- Отвечает на&nbsp;организационные вопросы: услуги, подготовка к&nbsp;анализам, как добраться
- Напоминает о&nbsp;визите за&nbsp;сутки и&nbsp;за&nbsp;час — сокращает no-show
- Переносит и&nbsp;отменяет запись без участия администратора
- Передаёт сложные вопросы живому сотруднику

Бот не&nbsp;консультирует по&nbsp;медицинским вопросам и&nbsp;не&nbsp;ставит диагнозы — только организация.

### Сайт клиники

- Услуги, врачи, лицензии — всё, что требует закон и&nbsp;ждёт пациент
- Онлайн-запись с&nbsp;первого экрана
- Быстрая загрузка с&nbsp;телефона — большинство пациентов приходят с&nbsp;мобильного
- Версия для слабовидящих
- SEO-структура под запросы «услуга + город»

### CRM и&nbsp;интеграция с&nbsp;МИС

- Единая база пациентов: откуда пришёл, история обращений, следующий визит
- Интеграция с&nbsp;МИС по&nbsp;API: IDENT, Medesk, Клиентикс, 1С:Медицина
- Автоматические напоминания о&nbsp;повторных приёмах и&nbsp;профилактике
- Хранение данных на&nbsp;серверах в&nbsp;России, согласия по&nbsp;152-ФЗ

### Реклама и&nbsp;репутация

- **Яндекс Директ** — с&nbsp;учётом требований к&nbsp;рекламе медицинских услуг
- **Яндекс Карты и&nbsp;2ГИС** — карточка, фото, работа с&nbsp;отзывами
- **ПроДокторов, НаПоправку** — профили врачей, ответы на&nbsp;отзывы
- **SEO** — статьи и&nbsp;страницы услуг, которые приводят пациентов из&nbsp;поиска бесплатно

### Аналитика

- Коллтрекинг: какой канал привёл звонок
- Сквозная аналитика: от&nbsp;клика до&nbsp;визита и&nbsp;повторного приёма
- Понятный отчёт раз в&nbsp;месяц: первичные, повторные, стоимость пациента по&nbsp;каналам

## Как работаем

1. **Аудит** — смотрим текущую запись, сайт, рекламу; считаем, где теряются пациенты
2. **Внедрение** — AI-администратор и&nbsp;запись за&nbsp;1–2&nbsp;недели, сайт за&nbsp;2–3&nbsp;недели
3. **Трафик** — Директ, Карты, SEO — когда есть куда вести и&nbsp;чем мерить
4. **Сопровождение** — оптимизация, отчёты, развитие без раздувания штата

## Почему клиники выбирают нас

- **Организация, а&nbsp;не&nbsp;медицина** — автоматизируем запись и&nbsp;маркетинг, не&nbsp;лезем в&nbsp;лечебный процесс
- **152-ФЗ по&nbsp;умолчанию** — серверы в&nbsp;России, согласия, ограничение доступа
- **Один подрядчик** — сайт, бот, CRM, реклама в&nbsp;одних руках, без «это не&nbsp;к&nbsp;нам»
- **Считаем деньги** — отчёты в&nbsp;пациентах и&nbsp;рублях, а&nbsp;не&nbsp;в&nbsp;кликах

## Частые вопросы

### Как AI-администратор работает с&nbsp;медицинскими данными?

Данные пациентов хранятся на&nbsp;серверах в&nbsp;России, собираются только с&nbsp;согласия, доступ ограничен. Бот не&nbsp;ставит диагнозы и&nbsp;не&nbsp;даёт медицинских рекомендаций — только запись, напоминания и&nbsp;организационные вопросы.

### Интегрируетесь с&nbsp;нашей МИС?

Работаем через API с&nbsp;IDENT, Medesk, Клиентикс, 1С:Медицина, Renovatio. Если у&nbsp;МИС есть API — интеграция возможна. Если нет — строим контур записи рядом, без замены МИС.

### Работаете с&nbsp;небольшими стоматологиями?

Да, основной профиль — малый и&nbsp;средний бизнес: стоматологии, косметология, многопрофильные центры на&nbsp;2–15&nbsp;кабинетов. Удалённо по&nbsp;всей России.

### Сколько занимает внедрение?

AI-администратор и&nbsp;связка с&nbsp;записью — 1–2&nbsp;недели. Сайт — 2–3&nbsp;недели. Рекламу запускаем, когда есть куда вести пациентов и&nbsp;чем измерять результат.
