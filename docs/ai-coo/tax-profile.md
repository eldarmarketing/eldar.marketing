# Налоговый профиль: ИП на УСН 6% «Доходы»

Режим подтверждён владельцем 2026-07-22. AI-COO ведёт финансовый контур по этому профилю. Перед любым расчётом сверять актуальность норм веб-поиском (законы меняются; НДС-порог снижается поэтапно).

## Параметры 2026

| Параметр | Значение |
|---|---|
| Ставка | 6% с дохода (кассовый метод — по поступлениям на счёт) |
| Лимит дохода УСН | 490,5 млн ₽/год |
| Фиксированные взносы ИП за себя | **57 390 ₽**, срок до **28.12.2026** |
| Доп. взнос | **1%** с дохода свыше 300 000 ₽, срок до **01.07.2027** (макс. 321 818 ₽) |
| Вычет взносов | Без сотрудников — налог до нуля; взносы «подлежащие уплате в этом году» учитываются даже до фактической оплаты. С сотрудниками — не более 50% |
| **НДС-порог** | **20 млн ₽** дохода → обязанность НДС (5%/7%) с 1-го числа следующего месяца. Далее: 15 млн (2027), 10 млн (2028) |

## Календарь (мониторит COO, напоминает заранее за 7 и 1 день)

| Срок | Событие |
|---|---|
| 25.04 | Декларация УСН за прошлый год (подаёт ИП с ЭЦП — COO готовит расчёт) |
| 28.04 | Налог УСН за прошлый год + аванс за 1 кв. (уведомление ЕНП до 25.04) |
| 01.07 | 1% взнос с превышения 300 тыс. за прошлый год |
| 28.07 | Аванс за полугодие (уведомление до 25.07) |
| 28.10 | Аванс за 9 месяцев (уведомление до 25.10) |
| 28.12 | Фиксированные взносы за себя |
| постоянно | Доход с начала года vs 20 млн ₽ (НДС) — алерт при 70/85/95% порога |

## Алгоритм расчёта аванса (COO делает автоматически, платит владелец)

1. Доход нарастающим итогом из выписок Точка API (только поступления от клиентов; исключить переводы себе, возвраты).
2. Налог = доход × 6% − взносы, подлежащие уплате в году − ранее уплаченные авансы.
3. Результат → владельцу: сумма, КБК/ЕНП, срок, черновик уведомления.
