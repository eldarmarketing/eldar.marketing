# Продуктизация: пакеты услуг Эльдар Маркетинг

Правило продуктизации: **фиксированная цена, фиксированный скоуп, стандартный workflow.** Всё, что вне скоупа — допсоглашение с отдельной сметой, не «ну ладно, сделаю».

> Цены ниже — вилки-драфт на утверждение Эльдару. Комплекс из всех пакетов калиброван по кейсу Руслана (70 000 ₽/мес).

## Пакет 1 · «Сайт+» — поддержка и развитие сайта

**Цена:** 25 000–35 000 ₽/мес · **SLA:** рутинная правка ≤ 48 ч

**Входит:**
- мониторинг работоспособности (healthcheck, uptime), SSL, бэкапы
- до 10 задач-правок в месяц (тексты, фото, баннеры, цены, блоки)
- обновление контента по материалам клиента
- техподдержка хостинга/домена, мелкие фиксы вёрстки

**Не входит:** новые разделы и функциональность, редизайн, интеграции (CRM, оплата), миграции — отдельная смета.

## Пакет 2 · «SEO-абонентка»

**Цена:** 30 000–40 000 ₽/мес · **Горизонт:** результат оценивается с 3-го месяца

**Входит:**
- техническая оптимизация (скорость, разметка, sitemap, индексация)
- семантика и структура: 2–4 новые/обновлённые страницы в месяц
- 2 SEO-текста в месяц (написание + размещение)
- ежемесячный отчёт: позиции, трафик из поиска, следующие шаги

**Не входит:** ссылочные бюджеты, гарантии «топ-3» (двигаем метрики, не обещаем позиции — п. 5.1 договора), контекстная реклама.

## Пакет 3 · «Директ-ведение»

**Цена:** 25 000–35 000 ₽/мес · **Рекламный бюджет клиента — отдельно, в пакет не входит**

**Входит:**
- ведение и оптимизация кампаний: ставки, минус-слова (еженедельно), аудитории
- до 2 новых кампаний/месяц, A/B заголовков и объявлений
- Метрика: цели, конверсии, сквозная картина по заявкам
- ежемесячный отчёт: расход, CPL, что меняем

**Не входит:** первичная настройка с нуля (разовая смета), видеокреативы, продвижение вне Яндекса.

## Пакет 4 · «Карты + контент»

**Цена:** 15 000–25 000 ₽/мес

**Входит:**
- Яндекс Карты / Бизнес: карточка, фото, услуги, цены — актуальность
- ответы на отзывы (в течение 48 ч), работа с рейтингом
- 4–6 публикаций в месяц (посты/новости на сайт или в соцсети по выбору)

**Не входит:** накрутка отзывов (не делаем), съёмка контента на месте.

## Комплекс «Всё включено»

Пакеты 1–4 вместе: **70 000 ₽/мес** (референс — договор № 1 с ИП Угурлуев). Скидка за комплекс ≈ 25–30% против суммы пакетов — фиксируем как стандарт.

## Workflow любого пакета (конвейер)

1. Заявка клиента → первая линия → GitHub Issues (борд «Клиенты», Inbox).
2. Оценка: рутина в скоупе → Todo; вне скоупа → «это допсоглашение», шаблон ответа №4.
3. Исполнение: рутина — помощник по SOP; производство — задача Claude Code по скиллу проекта.
4. Review (Эльдар) → отправка клиенту → Done.
5. Ежемесячно автоматом: счёт + акт (cron), отчёт по пакету.
