Сайт для автоателье BIRSON

Февраль 2026 · 2 недели

Задача

Клиент — автоателье BIRSON, специализируется на перетяжке салонов, покраске кожи и предпродажной подготовке автомобилей в Санкт-Петербурге. Два направления: премиальная перетяжка для владельцев и быстрая подготовка для перекупщиков.

Нужен был современный сайт, который передаёт премиальность услуг, показывает портфолио работ и конвертирует посетителей в заявки.

Решение

Стек: Next.js 16 + React 19 + Tailwind CSS + Prisma + PocketBase + Grammy (Telegram-бот).

Дизайн

  • Тёмная тема с золотыми акцентами — премиальный вид, ассоциация с кожей и люксом
  • Фоновое видео/фото текстуры кожи на hero-секции
  • Mobile-first: адаптивная верстка для всех устройств
  • CTA «Рассчитать стоимость» в шапке

Структура (10+ страниц)

  • Главная с hero-секцией, преимуществами и блоком для перекупщиков
  • Услуги с детальными страницами по каждому виду работ
  • Калькулятор стоимости — онлайн-расчёт без визита
  • Портфолио с фото работ
  • Цены
  • Отзывы
  • Блог
  • Контакты с Яндекс.Картой
  • Админ-панель
  • Политика конфиденциальности

Фичи

  • Калькулятор стоимости — клиент получает цену без звонка
  • Блок для перекупщиков — отдельная воронка: «Сфоткай → Отправь в Telegram → Получи цену за 10 минут»
  • ROI-калькулятор — «Вложи 5 000 ₽ → получи +20 000 ₽ к цене авто»
  • Telegram-бот — CRM для заявок и коммуникации
  • Портфолио через Telegram — мастер отправляет фото в бот, они появляются на сайте
  • S3-хранилище для медиафайлов
  • Генерация PDF (счета, реквизиты)
  • JSON-LD Schema для SEO
  • Яндекс.Метрика

Результат

  • Современный премиальный сайт, передающий качество услуг
  • Две воронки: для владельцев авто и для перекупщиков
  • Калькулятор снижает нагрузку на менеджера
  • Telegram-бот автоматизирует приём заявок и публикацию портфолио
  • Сайт работает на VPS (PM2 + Nginx + SSL)

Для кого

Автоателье, детейлинг-студии, автосервисы с премиальным позиционированием, бизнесы которым нужен калькулятор стоимости на сайте.
