Сайт для автосервиса DSG Clinic

Февраль 2026 · 2 недели

Задача

Клиент — DSG Clinic, специализированный сервис по ремонту коробок DSG, S-Tronic, PowerShift и АКПП в Санкт-Петербурге (ул. Ушинского, 12Н). Нужен был современный сайт, который вызывает доверие, показывает экспертизу и приводит клиентов.

Старый сайт: устаревший дизайн, нет мобильной адаптации, нет портфолио работ.

Решение

Стек: Next.js 16 + React 19 + Tailwind CSS 4 + PocketBase (отзывы).

Дизайн

  • Тёмная тема в стиле Vercel/Linear — техничный, премиальный вид
  • Монохромная палитра + акцентный цвет
  • Mobile-first: 70%+ трафика с мобильных
  • Sticky CTA-бар внизу: Звонок, Telegram, Записаться, Max

Структура (13+ страниц)

  • Главная с hero-видео, статистикой, каруселью портфолио
  • Страницы услуг (12 видов КПП: DSG7 DQ200, DSG6 DQ250, DSG7 DQ381, DQ500, PowerShift…)
  • Портфолио с видео ремонтов
  • Отзывы (48+ на Яндекс Картах, автоимпорт)
  • Контакты с Яндекс.Картой
  • Страница /qrkod (linktree-стиль для QR-кодов)
  • Акции · О нас · Цены · Марки авто

Фичи

  • SSG + ISR для максимальной скорости
  • JSON-LD Schema для SEO (Organization, LocalBusiness, Service)
  • Форма записи на диагностику
  • Интеграция с Telegram-ботом (@dsg_robot) — CRM для заказов
  • Яндекс.Метрика
  • Open Graph для соцсетей

Страница услуги

Портфолио с видео

  • Видео-карусель на главной
  • Каждая работа: фото до/после, описание, видео процесса
  • SSG-генерация страниц из базы данных

Мобильная версия

Контакты

Результат

  • Lighthouse: 95+ Performance, 100 SEO
  • Время загрузки: <1.5 сек
  • Конверсия в звонок/заявку выросла
  • 48+ отзывов интегрированы с Яндекс Карт
  • Сайт работает на VPS (PM2 + Nginx + SSL)

Для кого

Автосервисы, узкоспециализированные мастерские, любой бизнес которому нужен технологичный сайт с портфолио работ.
