B2B-каталог хозтоваров для госзаказчиков

Март 2026 · 2 недели

Задача

Клиент — оптовый поставщик хозяйственных товаров для ЖКХ, школ, больниц и государственных учреждений Санкт-Петербурга. Работает по 44-ФЗ и 223-ФЗ.

Нужен был современный каталог с удобным поиском, кодами КТРУ для госзакупок, формами запроса коммерческого предложения и блогом для SEO.

Решение

Стек: Next.js 15 + React 19 + Tailwind CSS + SSG (статическая генерация 500+ страниц).

Дизайн в стиле Госуслуг

  • Цветовая схема: синий #0D4CD3 + белый фон — институциональный, вызывающий доверие
  • Адаптивный дизайн для всех устройств
  • Мягкие тени, скруглённые кнопки, бейджи «Популярное» и «Новинка»
  • Sticky header с backdrop-blur

Структура (8+ разделов)

  • Главная с hero-секцией, поиском, счётчиками и FAQ
  • Каталог 400+ товаров с фильтрацией по категориям
  • Карточки товаров с кодами КТРУ и ОКПД
  • Раздел «Госзаказчикам»: инструкция, документы, преимущества
  • О компании с преимуществами и счётчиками
  • Блог с 6 SEO-статьями
  • Контакты с Яндекс.Картой
  • Политика конфиденциальности и пользовательское соглашение

Каталог

Фичи

  • 400+ товаров из двух источников (Nethouse + Galast) с унифицированным форматом
  • Коды КТРУ и ОКПД в каждой карточке товара — госзаказчик сразу видит нужные коды
  • Форма запроса КП — на каждой странице, с автоматической обработкой
  • Блок «Как заказать» — пошаговая инструкция для госзаказчиков
  • FAQ — ответы на частые вопросы об оплате, доставке, документах
  • Блок доверия — цифры, факты, опыт работы
  • S3-хранилище для 374 изображений товаров
  • JSON-LD Schema для SEO (Organization, Product, BreadcrumbList, BlogPosting)
  • Яндекс.Метрика

Для госзаказчиков

Блог

6 SEO-статей для привлечения органического трафика:

  • Как выбрать хозяйственный инвентарь для ЖКХ
  • Закупки по 44-ФЗ: пошаговая инструкция
  • Справочник кодов КТРУ для хозтоваров 2026
  • Подготовка к зимнему сезону
  • Как сэкономить на закупках
  • Документация при поставке товаров

Контакты

Мобильная версия

Юридическое соответствие (152-ФЗ)

  • Политика обработки персональных данных на отдельной странице
  • Пользовательское соглашение (цены не являются офертой)
  • Cookie-баннер с согласием (localStorage)
  • Обязательный чекбокс согласия под каждой формой
  • Ссылки в подвале сайта

Результат

  • SSG-сайт на 500+ статических страниц — мгновенная загрузка
  • Каталог с 400+ товарами, фильтрацией и поиском
  • Полное соответствие 152-ФЗ — защита от штрафов до 18 млн ₽
  • Блог для органического трафика из поисковых систем
  • Адаптивный дизайн в стиле госуслуг — доверие целевой аудитории

Для кого

Оптовые поставщики, работающие с госзакупками. Компании которым нужен каталог товаров с кодами КТРУ, соответствие законодательству и SEO-оптимизация.
