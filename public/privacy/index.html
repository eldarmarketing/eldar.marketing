Политика конфиденциальности

1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки персональных данных пользователей сайта eldarmarketing.ru (далее — Сайт).

Оператор: ИП, осуществляющий деятельность через сайт eldarmarketing.ru.

2. Какие данные собираются

При заполнении формы на Сайте пользователь предоставляет:

  • Имя
  • Телефон или Telegram
  • Описание задачи

3. Цели обработки

Персональные данные обрабатываются для:

  • Связи с пользователем по оставленной заявке
  • Обсуждения задачи и подготовки коммерческого предложения

4. Хранение и защита

  • Данные не передаются третьим лицам
  • Данные не используются для рассылок без согласия
  • Заявки обрабатываются через защищённый сервис FormSubmit
  • Данные хранятся до момента удаления по запросу пользователя

5. Права пользователя

Пользователь вправе:

  • Запросить информацию о хранимых данных
  • Потребовать удаления своих данных
  • Отозвать согласие на обработку

Для этого напишите: @eldarmarketing

6. Согласие

Отправляя форму на Сайте, пользователь подтверждает согласие с настоящей Политикой.
