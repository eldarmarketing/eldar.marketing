С 2015 года делаю маркетинг для бизнеса в Петербурге и удалённо по России. Четыре направления — можно по отдельности, можно всё сразу.

AI-ассистенты →

Собираю ботов на Claude API и GPT-4 — подключаю к Telegram, виджету на сайте. Бот отвечает клиентам, квалифицирует заявки, отправляет счёт в CRM. Работает 24/7 без участия менеджера. Последний проект — бот для автосервиса, который принимает заявки на ремонт и сам заносит их в Битрикс24.

Автоматизация →

Битрикс24, amoCRM, 1С — связываю между собой и с Telegram через n8n. Типичная задача: клиент оставляет заявку — она падает в CRM, менеджер получает пуш, клиент — подтверждение. Счета, отчёты, уведомления формируются сами.

Контекстная реклама →

Яндекс Директ, Telegram Ads, ВКонтакте, Авито. Для автосервиса в Петербурге снизил стоимость заявки с 1 200 до 780 рублей за два месяца ведения. Не беру проекты с бюджетом меньше 30 000 — при маленьком бюджете Директ не даёт статистики для нормальной работы.

Разработка сайтов →

Next.js + React + Tailwind CSS. Сделал сайты для DSG Clinic, BIRSON, Горец.Маркет, Товарка СПб — от автосервиса до B2B-каталога на 500+ страниц. Telegram-бот в комплекте: заявки, портфолио, уведомления. Готовый сайт за 7-14 дней.

Как это выглядит

Первый созвон — 30–40 минут, разбираемся где теряете деньги прямо сейчас. Через 3–10 дней уже работающий прототип. Отчитываюсь раз в неделю в Telegram — коротко, без PDF на 20 страниц.

Не беру больше 5 проектов одновременно. 11 лет в маркетинге, работаю напрямую с собственниками.
