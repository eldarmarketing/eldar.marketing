Разработка сайтов на Next.js в Санкт-Петербурге

11 лет в маркетинге, последние полгода строю сайты на Next.js. Не шаблоны из конструктора. Код на React, деплой на VPS, никакой абонентской платы.

Стек

Next.js + React + Tailwind CSS. Страницы генерируются статически — загрузка меньше 1.5 секунды. По моим проектам в нише автосервисов — 70%+ мобильного трафика, поэтому всё mobile-first.

JSON-LD Schema, Open Graph, Яндекс.Метрика, SSL. Деплой на VPS с Nginx — без посредников.

Проекты

DSG Clinic — сервис по ремонту коробок DSG в Петербурге. У клиента был сайт из 2018 года, без мобильной версии. Сделал за 2 недели: тёмная тема в стиле Vercel, 13 страниц, портфолио с видео ремонтов, автоимпорт 48 отзывов с Яндекс Карт. Telegram-бот работает как CRM — все заявки с сайта падают туда. Кейс →

У автоателье BIRSON другая история. Два типа клиентов — владельцы авто и перекупщики. Портфолио работает через Telegram — мастер снял видео, отправил в бот, оно на сайте. Кейс →

Горец.Маркет — интернет-магазин продуктов с Кавказа. Rails 7.2 + React 19 + Docker. Кастомная админка на 26 контроллеров: дашборд, заказы в модалках, массовое редактирование 150+ товаров. Отдельная история — 794 описания переписал нейросетью за 15 минут. Копирайтер делал бы это 2-3 недели. Кейс →

Товарка СПб — B2B-каталог для госзаказчиков. 400+ товаров с кодами КТРУ и ОКПД, SSG на 500+ статических страниц, соответствие 152-ФЗ. Кейс →

ТДО — поставщик продуктов HoReCa в Петербурге. 1500+ товаров, 2000+ клиентов. tdo.spb.ru

FreshFish — интернет-магазин рыбы и морепродуктов. Полный редизайн фронтенда в стиле ВкусВилл без единого изменения в бэкенде. freshfish.spb.ru

Telegram-бот в комплекте

Каждый мой сайт — это связка: сайт + бот + CRM. Бот принимает заявки, публикует портфолио, шлёт уведомления владельцу. Для бизнеса это управление с телефона, без входа в админку.

Не WordPress

Средний сайт на WordPress грузится 3-4 секунды. У DSG Clinic — Lighthouse Performance 95+, загрузка меньше 1.5 секунды. Код ваш, хостинг ваш, ограничений платформы нет.

Процесс

Первый созвон — 30-40 минут. Обычно клиент приходит с фразой «мне нужен сайт», а через 40 минут у нас структура на 10 страниц и понимание, какие заявки он хочет получать. Через 7-14 дней сайт работает.

Для кого

80% моих клиентов — авто-тематика и торговля. Но за 11 лет работал с 6+ нишами, в каждой есть готовый сайт в портфолио.
